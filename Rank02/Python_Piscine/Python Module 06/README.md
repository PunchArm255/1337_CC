# Python Module 06: Modules & Packages

This module covers the architecture of Python applications, focusing on the package system, module imports, and managing complex dependencies.

## Exercises

| Exercise | File(s) | Description |
| :--- | :--- | :--- |
| **ex00** | `alchemy/` | Designing a structured package with subpackages (`grimoire`, `transmutation`) and `__init__.py` files. |
| **ex01** | `ft_import_transmutation.py` | Exploring relative and absolute imports within a nested package structure. |
| **ex02** | `ft_circular_curse.py` | Identifying and resolving circular import issues in Python. |
| **ex03** | `ft_pathway_debate.py` | Understanding how Python finds modules and managing `sys.path`. |
| **ex04** | `ft_sacred_scroll.py` | Utilizing the complete `alchemy` package in a functional script. |
