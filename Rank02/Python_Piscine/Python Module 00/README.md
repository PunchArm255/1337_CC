# Python Module 00: Garden Basics

This module serves as an introduction to Python through basic concepts wrapped in a "Garden" theme. The exercises cover foundational topics such as printing, basic arithmetic, variable management, and simple logic.

## Exercises

| Exercise | File(s) | Description |
| :--- | :--- | :--- |
| **ex00** | `ft_hello_garden.py` | An introductory program to print a simple message (e.g., hello world equivalent). |
| **ex01** | `ft_plot_area.py` | Program that calculates and returns basic area functions, utilizing arithmetic operations. |
| **ex02** | `ft_harvest_total.py` | Simple script combining string printing and numeric variables. |
| **ex03** | `ft_plant_age.py` | Logic regarding tracking years or simple conditions. |
| **ex04** | `ft_water_reminder.py` | Introduction to condition statements (if/else) for reminders. |
| **ex05** | `ft_count_harvest_iterative.py`<br>`ft_count_harvest_recursive.py` | Deep dive into calculation loops, showcasing both iterative `for`/`while` structures and recursive implementations. |
| **ex06** | `ft_garden_summary.py` | Script formatting and presenting multiple data points cohesively. |
| **ex07** | `ft_seed_inventory.py` | Using simple data structures to manage an inventory. |
