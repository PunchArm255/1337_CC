# Python Module 07: Advanced OOP & Design Patterns

This module focuses on professional-grade Object-Oriented Programming, introducing common design patterns and complex inheritance structures through a card game simulation.

## Exercises

| Exercise | File(s) | Description |
| :--- | :--- | :--- |
| **ex00** | `Card.py`<br>`CreatureCard.py` | Implementation of base classes and specific entity types using inheritance. |
| **ex01** | `ArtifactCard.py`<br>`SpellCard.py`<br>`Deck.py` | Managing collections of objects and expanding the class hierarchy. |
| **ex02** | `Combatable.py`<br>`Magical.py`<br>`EliteCard.py` | Using multiple inheritance and mixins to add capabilities to classes. |
| **ex03** | `CardFactory.py`<br>`GameStrategy.py` | Implementing the Factory pattern for object creation and the Strategy pattern for behavior. |
| **ex04** | `TournamentPlatform.py`<br>`Rankable.py` | Developing an integrated system for managing tournaments and ranking entities. |
