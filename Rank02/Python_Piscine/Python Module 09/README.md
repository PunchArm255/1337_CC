# Python Module 09: Data Validation with Pydantic

This module focuses on ensuring data integrity and type safety in Python applications using the Pydantic library. The exercises cover model definition, validation logic, and the use of Enums.

## Exercises

| Exercise | File(s) | Description |
| :--- | :--- | :--- |
| **ex00** | `space_station.py` | Building robust data models with Pydantic and implementing basic field validation. |
| **ex01** | `alien_contact.py` | Utilizing Enums and complex model validators to ensure data consistency. |
| **ex02** | `space_crew.py` | Developing nested data structures and performing deep validation on complex objects. |
