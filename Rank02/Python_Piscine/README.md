# Python Piscine

Welcome to the Python Piscine (because fuck C)! This repository contains a series of modules designed to teach Python programming from the ground up, starting with basic syntax and progressing to advanced concepts.

## Modules Overview

Here is a summary of the modules covered in this Piscine:

| Module | Status | Description |
| :--- | :---: | :--- |
| **[Python Module 00](./Python%20Module%2000)** | ✅ Completed | Introduction to Python syntax, basic data structures, and simple scripts. Emphasizes "Garden Basics" with examples spanning basic output, calculations, and loops. |
| **[Python Module 01](./Python%20Module%2001)** | ✅ Completed | Fundamentals of Object-Oriented Programming (OOP). Introduces classes, objects, inheritance, and basic garden data management. |
| **[Python Module 02](./Python%20Module%2002)** | ✅ Completed | Deep dive into Exception Handling. Covers generic exceptions, custom errors, and handling failures gracefully (using try, except, else, finally blocks). |
| **[Python Module 03](./Python%20Module%2003)** | ✅ Completed | Introduction to more complex structures, including game systems, achievement trackers, data streams, and coordinate systems. |
| **[Python Module 04](./Python%20Module%2004)** | ✅ Completed | Exploration of File I/O, data manipulation, and basic security measures for storage systems. |
| **[Python Module 05](./Python%20Module%2005)** | ✅ Completed | Advanced architectural patterns using Abstract Base Classes (ABC), polymorphism, and data processing pipelines. |
| **[Python Module 06](./Python%20Module%2006)** | ✅ Completed | Deep dive into Python's module and package system, including subpackages and circular dependency management. |
| **[Python Module 07](./Python%20Module%2007)** | ✅ Completed | Advanced Object-Oriented Programming and Design Patterns, featuring the Factory and Strategy patterns. |
| **[Python Module 08](./Python%20Module%2008)** | ✅ Completed | Professional project setup, including dependency management with `pyproject.toml` and environment configuration. |
| **[Python Module 09](./Python%20Module%2009)** | ✅ Completed | Data validation and type safety using Pydantic models, Enums, and complex model validators. |
| **[Python Module 10](./Python%20Module%2010)** | ✅ Completed | Functional programming concepts in Python, including Lambdas, Higher-order functions, and Decorators. |

Each module directory contains its own `README.md` with specific details about the exercises included. Navigate to the respective module folder to view instructions and the code.
