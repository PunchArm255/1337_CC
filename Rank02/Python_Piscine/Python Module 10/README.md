# Python Module 10: Functional Programming & Decorators

This final module explores the functional side of Python, covering lambdas, higher-order functions, scope management, and the powerful decorator pattern.

## Exercises

| Exercise | File(s) | Description |
| :--- | :--- | :--- |
| **ex00** | `lambda_spells.py` | Writing concise logic using anonymous (lambda) functions. |
| **ex01** | `higher_magic.py` | Applying functional tools like `map`, `filter`, and `reduce` to process data. |
| **ex02** | `scope_mysteries.py` | Understanding variable scope (LEGB) and implementing closures. |
| **ex03** | `functools_artifacts.py` | Utilizing the `functools` module for partial functions, caching, and more. |
| **ex04** | `decorator_mastery.py` | Designing and applying custom decorators to enhance function behavior. |
