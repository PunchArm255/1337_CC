# Python Module 04: Data Manipulation & I/O

This module explores Python's capabilities for handling files, managing data streams, and implementing basic security for stored information. The exercises focus on reading and writing files, processing binary or text data, and handling data integrity.

## Exercises

| Exercise | File(s) | Description |
| :--- | :--- | :--- |
| **ex00** | `ft_ancient_text.py` | Introduction to reading files and handling missing file exceptions. |
| **ex01** | `ft_archive_creation.py` | Program that demonstrates writing data to files and creating new archives. |
| **ex02** | `ft_stream_management.py` | Managing data streams and implementing efficient read/write operations. |
| **ex03** | `ft_vault_security.py` | Implementing basic security measures for sensitive data stored in files. |
| **ex04** | `ft_crisis_response.py` | Developing a comprehensive logging and response system for data integrity issues. |
