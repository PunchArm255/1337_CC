# Python Module 05: Pipelines & Abstract Classes

This module focuses on advanced Object-Oriented Programming concepts in Python, specifically Abstract Base Classes (ABC), polymorphism, and the creation of extensible data processing pipelines.

## Exercises

| Exercise | File(s) | Description |
| :--- | :--- | :--- |
| **ex00** | `stream_processor.py` | Implementation of Abstract Base Classes to define a foundation for various data processors. |
| **ex01** | `data_stream.py` | Building polymorphic stream handlers for environmental, financial, and system event data. |
| **ex02** | `nexus_pipeline.py` | Creating a consolidated pipeline to manage and filter multiple data streams through a unified interface. |
